What is virtualization?

Virtualization it is a virtual version from the something physical, including computer hardware platforms, operating systems, storage devices, computer network resources, and etcetera etcetera And there is some types of virtualization, like network virtualization, memory virtualization, grid computing, application virtualization, storage virtualization, server virtualization. But in this section i just explain about server virtualization
So in server virtualization it is just like that, we can create virtual machine or server inside on the real server
and in server virtualization there is 3 technique,

3 Techniques of Virtualization

Full Virtualization

Para-Virtualization

OS Level Virtualization

Full virtualization In this technique will uses a software that can create virtual machines called Hypervisor. And the hypervisor will interacts directly with the physical server's CPU and disk space. So the virtual machines "does not aware" that they are actually just virtual not physical.

Full Virtualization

Para Virtualization

Para-Virtualization Para-virtualization it is a litte different than the full virtualization technique. the different it is the virtual machines kernel is modified and then the virtual machines using this technique it is "aware" that they are just virtual.

OS Level Virtualization

OS Level Virtualization In this technique, doesn't require hypervisor at all, because the virtualization capability is part of the host OS. Each virtual machines is independent of one another. As guests machine must remain the same OS with the host machine, this is called a homogeneous environment.

2 Types of Hypervisor

Hypervisor Type 1 (Baremetal)

Hypervisor Type 2 (Hosted)

And in the Hypervisor, there is 2 types of that the first is Hypervisor type 1 or we can called baremetal and the second it is Hypevisor type 2 or we can called hosted
How hypervisor baremetal works? it is work doesn't require an operating system to run the hypervisor, because the hypervisor it self it can be an operating system.
And the hosted hypervisor it is the opposite of the baremetal, because hosted hypervisor require an operating system to install and run the hypervisor, the operating system it is like windows, or MacOS, or Linux etc..
